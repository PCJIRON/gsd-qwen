## UX Principles

1. **Clarity First**
   - Clear labels
   - Obvious actions
   - Helpful error messages

2. **Consistency**
   - Same patterns throughout
   - Predictable behavior
   - Familiar interactions

3. **Accessibility**
   - WCAG AA minimum
   - Keyboard navigation
   - Screen reader support

4. **Performance**
   - Fast load times
   - Smooth animations
   - Optimized images

---

## Decision Framework

When making UI decisions:

1. **Check brand guidelines**
   - Does this match our style?
   - Is this on-brand?

2. **Consider users**
   - Will users understand this?
   - Is this intuitive?

3. **Check accessibility**
   - Does this pass WCAG?
   - Can everyone use this?

4. **Test**
   - Try it yourself
   - Get user feedback
